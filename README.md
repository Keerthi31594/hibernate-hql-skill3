# Hibernate HQL Skill 3 - Sorting, Pagination & Aggregates

## Overview
This project demonstrates advanced HQL operations using the Product entity:
- Sorting (ASC/DESC)
- Pagination
- Aggregates (COUNT, MIN, MAX, AVG, SUM)
- GROUP BY operations
- Filtering with WHERE
- Pattern matching using LIKE

## Technologies
- Java 8+
- Hibernate 5.x
- MySQL/PostgreSQL
- Maven

## Setup
1. Clone the repository
2. Update database credentials in `hibernate.cfg.xml`
3. Run `ProductDataLoader` to populate sample data
4. Execute queries from `HQLDemo` class

## Tasks Completed
- Product entity setup
- Sample data insertion
- Sorting by price and quantity
- Pagination implementation
- Aggregate operations
- GROUP BY queries
- Price range filtering
- LIKE pattern matching
